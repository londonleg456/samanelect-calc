점유율 산정 기준

여러 전선을 동일 관내에 넣을 때 전선 피복(절연물 포함) 완성단면적의 총합 ≤ 전선관 내단면적의 1/3(약 33.3%)을 적용합니다. 굵기 구분 없이 통일된 현행 권장 기준입니다. (내선규정 전선관 굵기 선정, KEC 232 금속관·합성수지관·가요전선관 공사)

전선관 내경

후강전선관(KS C 8401) 호칭 16~104은 근사 내경(짝수)으로 실제 KS 내경값을 사용합니다. ELP전선관(KS C 8455, 파상형 경질 폴리에틸렌·지중용)은 호칭이 곧 내경(30~200mm) 점유율은 호칭 16~104의 KS 내경값을 사용합니다.

전선·케이블 외경

단심 절연전선은 HFIX(KS C 3341)와 F-GV 접지선을 사용합니다. F-CV 전력케이블(KS C IEC 60502-1)은 1심(단심)부터 다심까지 선택할 수 있으며, F-CVV 제어케이블, 22.9kV CN-CV, 소방용 F-FR-8 내화케이블 데이터가 포함되어 있습니다. F-CV는 난연(난연 PVC 시스) 처리만 다를 뿐 CV와 동일 규격·치수이므로 단심 CV는 F-CV 1심으로 통합했습니다. 표에 없는 규격은 '직접입력' 모드를 사용하세요.

참고 · 본 계산은 점유율(면적) 기준의 1차 산정 도구입니다. 가닥수가 적을 때(2~3본)는 기하학적 외접원 보정으로 필요 관경이 달라질 수 있습니다. 최종 설계 시에는 현장 조건을 함께 검토하세요.

⚡ 전선관 관경 계산기 사용 설명서

관경 계산기는 입력한 전선의 종류·굵기·가닥수(본수)를 바탕으로 점유율 33.3% 기준에 부합하는 최소 전선관 규격을 자동으로 산정해 주는 도구입니다.

🚀 사용 방법

1. 전선관 종류 선택

계산할 전선관의 재질·용도에 맞게 하나를 선택합니다.

  • 후강전선관 · 일반적인 금속 강관 (KS C 8401)
  • ELP전선관 · 파상형 경질 폴리에틸렌관, 주로 지중선로용 (KS C 8455)
  • HI-PVC전선관 · 내충격성 경질 비닐 전선관 (KS C 8431)

2. 전선 정보 입력

  • 종류 · HFIX, F-GV(접지선), F-CV(전력), F-FR-8(내화), 통신선 등 목록에서 선택
  • 규격 · 선택한 종류에 맞는 규격(단면적 ㎟, 코어 수 등)을 선택
  • 본수 · 해당 규격의 전선 가닥수를 숫자로 입력
  • 추가·삭제 · 여러 종류가 혼합되면 + 전선 추가로 행을 늘리고, 우측 ×로 삭제

💡 목록에 없는 규격이나 특수 케이블은 '직접입력(외경)'을 선택해 케이블 바깥지름(mm)을 직접 넣을 수 있습니다.

3. 결과 확인

전선 정보를 입력하면 즉시 상단 패널과 하단 표에 결과가 업데이트됩니다.

📊 요약 결과 패널

  • 추천 최소 전선관 · 전선 완성단면적 합계가 전선관 내단면적의 33.3%를 넘지 않는 가장 경제적인(작은) 호칭을 제시
  • 상태 표시
    • ● 적합 · 추천 규격이 존재하며 점유율 기준 만족
    • ● 기준 초과 · 가장 큰 규격을 써도 33.3%를 초과 (배관을 나누거나 더 큰 규격 필요)
  • 단면 개념도 · 바깥 원 = 전선관, 색칠된 원 = 전선 면적 합계, 점선 = 허용 한계(33.3%)

📋 규격별 검토 표

  • 선택한 전선관 종류의 모든 호칭에 대한 내경·단면적·허용면적(33.3%)·실제 점유율을 한눈에 확인
  • 우측 판정 열로 어떤 규격부터 '적합'한지, 어떤 규격이 최적의 '추천'인지 비교 검토

📌 산정 기준 및 주의사항 (필독)

본 계산기의 결과는 현행 규정에 따른 1차 산정 및 검토용입니다.

1. 산정 근거 (점유율 기준)

  • KEC 232 및 내선규정 · 동일 관내에 여러 전선을 넣을 때, 피복(절연물 포함) 완성단면적 총합이 전선관 내단면적의 1/3(약 33.3%) 이하가 되어야 합니다. 본 계산기는 이 기준을 엄격히 적용합니다.
  • 데이터 출처 · 전선관 내경 및 전선 외경은 각 품목 KS 규격(KS C 8401, KS C 8431, KS C 3341, KS C IEC 60502-1 등)을 기반으로 작성

2. 설계 시 주의사항 (기하학적 한계)

이 도구는 단순히 '면적의 합'으로 점유율을 계산합니다.

가닥수가 매우 적을 때(예: 굵은 케이블 2~3가닥)는 전선 배치의 기하학적 형태(외접원)로 인해, 면적 점유율이 33.3% 미만이어도 실제로는 관을 통과하지 못하거나 마찰이 심할 수 있습니다.

따라서 최종 실시설계·시공 시에는 반드시 현장 조건과 기하학적 여유(외접원 보정 등)를 추가 검토해야 합니다. 본 결과는 법적 효력을 갖는 최종 설계서가 아닙니다.
